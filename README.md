# Xophz Pegasus Boots

> **Category:** Wizard's Tower · **Version:** 1.0.0

Equip your site with the speed of flight with this theme and page builder.

## Description

**Pegasus Boots** is a planned visual theme and page builder for the COMPASS platform. It aims to accelerate design and content creation processes by providing a modern, drag-and-drop toolset that natively embraces the COMPASS design system and aesthetic.

## Requirements

- **Xophz COMPASS** parent plugin (active)
- WordPress 5.8+, PHP 7.4+

## Status

🔴 **In Development** - Currently scaffolded with base WordPress plugin Architecture. The page building interface and theme generation logic have not yet been implemented.

## Installation

1. Ensure **Xophz COMPASS** is installed and active.
2. Upload `xophz-compass-pegasus-boots` to `/wp-content/plugins/`.
3. Activate through the Plugins menu.
4. Access via the COMPASS dashboard → **Pegasus Boots**.
